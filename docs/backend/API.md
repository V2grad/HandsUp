# Intro

APIs are classified by Services

# Content 
- Authentication (non-standard)
- User
- Event
- Request

## Auth
- GET (validation)
- POST (registration)

## User
- POST (update user)
- DELETE (remove user)

## Event
- GET: 
"/:id" (get ID); 
"/view/:id" (view event)
- PUT (create event)
- POST: 
"/:id" (update event)
"/join/:id" (join a user)
- DELETE:
"/:id" (delete an event)
